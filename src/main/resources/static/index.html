투자 판단 분석 대시보드

포트폴리오 현황, 차트, 투자 판단 점수, 놓친 기회/피한 위험, 섹터 진단, API 상태를 페이지별로 확인할 수 있는 정적 프론트입니다.

대시보드

총 평가금액, 손익, 보유 종목, 섹터 비중을 한 화면에서 봅니다.

차트 분석

종목별 캔들차트와 거래량을 조회하고 URL 파라미터로 종목 이동을 받습니다.

투자 판단 평가

진입, 청산, 리스크, 분산, 섹터 적합성 점수를 확인합니다.

기회 / 위험

놓친 기회, 피한 위험, 너무 빨리 판 거래, 오래 보유한 거래를 확인합니다.

섹터 진단

강한 섹터와 약한 섹터, 그리고 현재 포트폴리오 노출을 진단합니다.

API 테스트

핵심 API 응답을 브라우저에서 한 번에 호출해 정상 동작 여부를 확인합니다.
